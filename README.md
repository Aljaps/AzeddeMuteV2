# AzeddeMute
> One key. Mic muted. Done.

Simple microphone control for Windows — built for gamers, streamers, and anyone stuck in back-to-back meetings.

---

## Download
**[→ AzeddeMute v2.0.0 (.zip)](../../releases/latest)**  
No installer. No setup. Just run the `.exe`.

---

## What it does

| Feature | Description |
|---|---|
| Toggle Mute | Click the mic or hit your hotkey to mute/unmute instantly |
| Push-to-Talk | Hold = mic on, release = mic off |
| Global Hotkey | Works even when you're in a game or browser |
| System Tray | Runs quietly in the background — red = muted, green = live |
| Custom Shortcut | Set any key you want 

---

## Usage

1. Run `AzeddeMute.exe`
2. Click **Setup Shortcut** → press your key → OK
3. Use the hotkey or click the mic to toggle
4. Minimize to tray with the yellow button when done

---

## Notes

- When the app exits, your mic is automatically **unmuted**
- Works on **Windows 10 / 11 x64** only
- No .NET installation required — runtime is bundled
